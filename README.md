# Gooddata JS SDK

## Usage

* npm install in repo
* run `gulp` in repo root
* built library is ready for you `build/gd-sdk.min.js`
* use with dependencies (jQuery 1.9.x or 2.x and [jQuery CSV](https://code.google.com/p/jquery-csv/))
